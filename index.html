Euromillions Predictions

Fun project for learning React and get some hands on with array methods. The goal was to increase the odds of the game by discarding all sequential number combinations, all even or all odd, and a few other constrains. I managed to reduce the 2.118.760 possible combinations for the numbers, to less than 800.000. I also added a countdown for the next draw, that happens every Tuesday and Friday at 19:45 (UTC).

The calculations for the results took about 15 seconds in a desktop, so I opted to dump the results on about 40 (1Mb) .json files and feed a random one on every request.

  • Responsabilities: Design, back-end, front-end
  • Technologies used: Javascript, CSS
  • Frameworks: ReactJS and styled-components
  • Website: Euromillions
  • GitHub: Repository
